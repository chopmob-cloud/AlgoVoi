AlgoVoi-Hand
Compliance Summary

AlgoVoi-Hand operates as an x402 payment gateway. This page summarises its alignment with UK PSR 2017/PSD2 and related frameworks. Full audit documentation is available on request under NDA.

Key Alignments

🔐
No Custody
Crypto settles direct on-chain. No fiat funds are held or pooled by AlgoVoi-Hand at any point.
✍️
SCA via x402
Wallet signatures provide strong customer authentication with dynamic linking — amount, payee, and token are bound to every signature.
🏗️
Tenant Isolation
5-layer row-level security with opaque tokens. Payment data, API keys, and receipts are fully isolated per tenant at the database level.
🛡️
Security
TLS 1.3, MFA enforcement, immutable audit logs, and per-tenant kill switches. Security is enforced at the gateway, not just documented.
🪪
AML
Structured onboarding with identity verification. No anonymous merchant access.
📋
Audit Trail
Every challenge, verification, limit change, and key event is logged per tenant with typed event codes and no PII in log lines.

Framework Scope Status
